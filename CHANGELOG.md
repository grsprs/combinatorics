## [1.0.0] - 2026-01-09

### Added
- Factorial package with efficient big.Int computation
- Binomial coefficient package with optimized nCr calculation
- Permutations package with Heap's algorithm and iterator support
- Combinations package with k-combinations and iterator support
- Comprehensive test suite with 100% code coverage
- Benchmarks for all core functions
- Full documentation (README, CONTRIBUTING, SECURITY, CODE_OF_CONDUCT)
- CI/CD pipeline with GitHub Actions
- GoDoc examples for all packages
- Generic type support for permutations and combinations
